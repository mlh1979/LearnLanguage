# Dictionary Data Attribution

This application uses dictionary data from the following sources:

## Wiktionary
Dictionary definitions and translations are derived from Wiktionary (https://www.wiktionary.org/),
licensed under the Creative Commons Attribution-ShareAlike 4.0 International License (CC-BY-SA 4.0).
https://creativecommons.org/licenses/by-sa/4.0/

Wiktionary data is used for English word definitions and translations into multiple languages.

## Google Translate
Word translations are obtained via Google Translate API for demonstration purposes.

## Dictionary API
English word definitions are sourced from the free Dictionary API
(https://dictionaryapi.dev/), which aggregates data from WordNet and Wiktionary.
